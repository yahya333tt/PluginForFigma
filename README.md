Автоматизированный конфигуратор стримера (Figma Plugin)

Плагин для Figma, который автоматически создаёт презентацию-сравнение периферии для стримера на основе загруженного JSON-файла.

Что делает плагин

1. Открывает небольшое окно 
2. Просит загрузить JSON-файл с категориями устройств  
3. Создаёт в текущей странице Figma:  
   - Титульный слайд  
   - По одному слайду на каждую категорию (с большой таблицей сравнения)  
   - Итоговый слайд с выбранной конфигурацией, общей стоимостью и списком ПО

Как запустить плагин

\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\

- Установлен **Figma Desktop**
- Установлен **Node.js**  https://nodejs.org
- Установлен **Visual Studio Code**  https://code.visualstudio.com

\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\

Запуск плагина в Figma

Откройте Figma Desktop
Создайте новый файл или откройте любой существующий

В меню выберите:
Plugins → Development → New Plugin

Придумайте имя для плагина,
Выберите Multi-product
Выберите Custom UI
Выбираете куда сохранить вашу папку и можете сразу её открыть

Далее:
Заменяете код в code.ts и ui.html

Откройте встроенный терминал в VSC

!!!ВАЖНО!!! Проверьте правильный ли у вас путь в терминале! Если нет, то "cd 'вашпуть'"

Выполните команду:

npm install

npm install -g typescript

npm install --save-dev @figma/plugin-typing

npm run watch

После этого ваш плагин уже доступен в figma и вы можете

Возможные проблемы и решения

Ошибка "This plugin template uses TypeScript..." → Не запущен npm run watch или code.js не сгенерировался. Запустите watch и сохраните code.ts.

Плагин не появляется в меню → Перезапустите Figma полностью.
